# SpeedAuto - Front-End

## Descrição
Este repositório contém o **front-end do SpeedAuto**, sistema SaaS para gestão de concessionárias de veículos.  
A interface é construída em **React.js** com **Tailwind CSS**, oferecendo dashboards, cadastro de veículos e clientes, registro de vendas e visualização de estatísticas em tempo real.  

O front-end consome a **API RESTful** do back-end (`speedauto-back-end`) para exibir e manipular dados.

---

## Tecnologias Utilizadas
- **React.js** — Biblioteca para construção da interface.  
- **Tailwind CSS** — Framework CSS para estilização responsiva.  
- **React Router** — Navegação entre páginas.  
- **Axios/Fetch** — Para consumo da API RESTful.  
- **Recharts** — Para gráficos e dashboards.
