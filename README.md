## 📖 Sobre o Projeto

**SpeedAuto** é uma plataforma de gestão completa para concessionárias de veículos, desenvolvida para transformar dados confusos em decisões rápidas e colocar sua concessionária na rota do crescimento acelerado.

> *"Onde a gestão corre na mesma velocidade dos seus negócios."*

A lentidão dos sistemas antigos e o caos das planilhas estão freando os lucros das concessionárias. A SpeedAuto nasceu para ser o motor da sua gestão.

## ✨ Funcionalidades

- 📊 **Dashboard em Tempo Real** - Visão completa do negócio
- 🚗 **Estoque de Veículos** - Gerenciamento completo do inventário
- 💰 **Gerenciamento de Vendas** - Acompanhamento de transações e faturamento
- 👥 **Leads** - Controle de clientes e oportunidades
- 💳 **Simulador de Financiamento** - Cálculo transparente de parcelas
- 📈 **Relatórios e Análises** - Dados acionáveis e exportáveis

## 🚀 Como Usar

### Instalação Local

```bash
# Clone o repositório
git clone https://github.com/vt-debug/speedauto-HTML-fd.git

# Entre no diretório
cd speedauto-HTML-fd

# Abra o index.html no navegador
# ou use um servidor local como Live Server
```

### Deploy

🌐 **Acesse a versão online:** [speedauto.vercel.app](https://speedauto-dun.vercel.app/)
